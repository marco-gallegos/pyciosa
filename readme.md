## Modulos

### Periodo

Define una serie de funciones para el manejo de periodos en formato usado internamente, es decir : YYYYMM -> 202001

#### Lista de Funciones

 * is_valid
 * explode
 * get_periodo
 * interpolate
 * get_previous_quarter_periodo
 * get_periodo_label
 * get_periodo_range_label
 * validate_date
 * get_period_full_label
